# 05 | 练习：探索 MCP 生态

## 🎯 练习目标

- 了解官方 MCP Server 生态
- 掌握安装和配置 MCP Server 的方法
- 理解 MCP 的实际应用场景

---

## 📋 练习要求

### 基础任务（必做）

**安装并配置一个官方 MCP Server**（在 Claude Desktop 中）：

1. 选择一个你感兴趣的官方 Server
2. 在 Claude Desktop 的配置文件中添加该 Server
3. 用自然语言让 Claude 调用该 Server 的工具
4. 记录：调用了什么工具？AI 返回了什么？

推荐从 `filesystem` 或 `github` 开始。

### 进阶任务（选做）

**设计一个你工作/学习中需要的 MCP Server**：

- 这个 Server 提供什么工具？
- 解决什么问题？
- 目标用户是谁？

不需要写完整代码，只需要设计思路。
